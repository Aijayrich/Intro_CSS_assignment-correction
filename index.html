What I Have Learned so Far

The importance of CSS really can't be understated. It is very important as it controls all design-related aspects of your website. What's more, CSS allows you to control and adjust the design and formatting of your entire website from a single spot. It is also of three types. The inline, internal and the external. Inline referss to HTML element having styling (CSS) element In internal CSS, you have to make use of the style tag. It is defined in the 'head' section of an HTML page, within a 'style' element. External CSS must not contain any HTML code and must be saved with a .css extension. The href attribute is where you define the path of the external css file. This method can be used by more than one html page. To use this method, add a link to it in the 'head' section of each html page.

The Difference between Paddings, Margins, and Borders

Padding is the space between the content and the border, whereas margin is the space outside the border. Border is a line that surrounds the element.
